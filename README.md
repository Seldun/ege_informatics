# Решения задач ЕГЭ по информатике на Python

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Проект содержит решения всех типов задач ЕГЭ по информатике с подробными комментариями.

## Структура проекта
```
ege/
├── type_01/        # Задание 1: Кодирование информации
├── type_02/        # Задание 2: Таблицы истинности
├── ...
└── type_27/        # Задание 27: Программирование
```

## Как использовать
1. Клонировать репозиторий:
```bash
git clone https://github.com/Seldun/ege_informatics.git
cd ege_informatics
```

2. Установить зависимости:
```bash
python -m venv venv
source venv/bin/activate  # Linux/MacOS
venv\Scripts\activate    # Windows
pip install -r requirements.txt
```

3. Запустить решение:
```bash
python ege/type_01/solutions/task_01.py
```
