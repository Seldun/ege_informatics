краткий и универсальный код для перевода числа из десятичной системы в систему с основанием до 36:

```python
def to_base(n, base):
    s = ''
    while n > 0:
        n, r = divmod(n, base)
        s = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'[r] + s
    return s or '0'
```
Примеры использования:
```python
print(to_base(42, 13))   # → "33"
print(to_base(255, 16))  # → "FF"
print(to_base(0, 12))    # → "0"
```
Как это работает:
```divmod(n, base)``` возвращает частное и остаток от деления n на base.

Остаток r определяет символ из строки-алфавита (0-9, A-Z).

Символ добавляется в начало строки s.

Процесс повторяется, пока n > 0.

Если исходное число 0, возвращается '0'
